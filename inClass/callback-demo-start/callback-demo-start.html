  1. This example demos one way for a component to call methods that are defined in another JS script
  2. In this case, we want to have the Hello and Goodbye buttons of the <app-header> component call a specific method in main.js - buttonCallback()
  3. main.js is all done for you, so what you need to do is to go ahead and modify AppHeader so that
    1. Clicking #btn-hello invokes the callback function and passes in (to that function) the greeting property of the component
    2. Clicking #btn-goodbye invokes the callback function and passes in (to that function) the farewell and specialNumber properties of the component
